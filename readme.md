# Plataforma de Eventos

Este projeto foi desenvolvido na disciplina  seminário integrador, com o objetivo de criar uma aplicação web focada no desenvolvimento Front-End. Durante o curso, foram aplicados os conhecimentos adquiridos para criar uma solução prática que permite aos usuários interagir com a plataforma de forma intuitiva e eficiente.

# Objetivo do Projeto

O principal objetivo deste projeto é desenvolver uma plataforma de eventos colaborativa, onde os próprios usuários podem criar, gerenciar e divulgar seus eventos. A aplicação foi projetada com foco em uma interface intuitiva e acessível, oferecendo uma experiência de navegação agradável e eficiente. Nesta primeira etapa, o desenvolvimento concentrou-se na construção do Front-End da plataforma.
